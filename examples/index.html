jQuery.currency Demo

€ 123.45 EUR
£ 543.21 GBP

Usage:

Expected microformat is like: 


<abbr class="unit">&euro;</abbr> <span class="amount">123.45</span> <abbr class="currency">EUR</abbr>

Configure once, then run as many times you want: 


jQuery.currency.configure({
  // Configure rates and default settings
  rates: {
    // Object containing exchange rates from the base currency (EUR by default)
    "USD": 1.2491,
    "GBP": 0.8032
  }
});

jQuery( ... ).currency("USD"); // Convert to USD
